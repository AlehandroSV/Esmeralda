# Esmerald CLI

CLI for [Jade](https://github.com/user/jade) ORM.

## Installation

```bash
npm install -g esmerald
```

## Usage

```bash
esmerald init          # Scaffold a new Jade project
esmerald generate      # Generate migration from schema diff
esmerald migrate       # Run pending migrations
esmerald db pull       # Introspect DB → entity files
esmerald db push       # Push schema directly to DB
```
